# 项目概述
    本项目由简单到复杂逐步实现了一个简单的基于epoll的多线程C++服务器，能够接受多个客户端连接并处理客户端请求。项目采用现代C++编写，并使用spdlog库进行日志记录。介于能力有限，目前只实现了基础版本。后续会逐步扩充与优化设计。
